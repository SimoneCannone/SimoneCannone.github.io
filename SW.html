About the Project

Our game is heavily inspired by Lethal League but with a unique twist: it relies solely on physics, making it more unpredictable and chaotic. The core of the game revolves around a chaotic arena where players compete against each other, with the physics adding an extra layer of excitement and challenge. The unpredictable movement and interactions make every match feel fresh and intense.

Main Tasks

In this project, my main responsibilities included developing a complete user interface system from scratch. I implemented a 2D image system, interactive buttons, and text rendering, which provided a solid foundation for building a functional and visually consistent UI throughout the game.

In addition to the interface system, I designed and implemented the entire game UI, ensuring seamless integration with the internal logic and a smooth user experience. I also developed the audio system used in the project, handling the playback of sound effects and music in an efficient and customizable way.

Main Features

PS5 Game

Developed a next-generation game tailored for PlayStation 5, taking advantage of its hardware capabilities with high-resolution graphics, fast loading times, and immersive features.

Custom Engine

Built a custom game engine from scratch, including a modular architecture and a dynamic leveling system that allows players to gain experience and unlock unique abilities.

Scene System

Implemented a custom scene management system within the engine, allowing for seamless transitions, efficient resource handling, and modular game state control.

Animations and 3D Models

Created and integrated optimized 3D models and animations, ensuring visual quality and performance across a variety of Android devices.
