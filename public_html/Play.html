About the Project

The Play Engine is a lightweight and efficient game engine built to make handling 3D models, textures, audio, and resource management as smooth as possible. Designed for flexibility and high performance, it provides a solid foundation for game developers who want a modular and scalable framework without unnecessary complexity.

Main Tasks

Seamless 3D Model Loading with TinyObjLoader. The engine uses TinyObjLoader to load and render 3D models in the OBJ format effortlessly. It efficiently manages vertex buffers, normal and UV mappings, and memory allocation, ensuring smooth handling of large and complex assets. The rendering pipeline is designed to minimize CPU load while making the most of GPU acceleration.

Optimized Texture Loading in GNF Format. Play Engine supports textures in the GNF (Graphics Native Format), which is optimized for graphical assets. It includes automatic mipmap generation, anisotropic filtering, and smart VRAM allocation to keep rendering smooth and efficient.

Real-Time Audio Playback with Audio Out 2. Audio is a crucial part of any immersive experience, and Play Engine integrates Audio Out 2 for real-time playback. It supports multi-channel audio, spatial sound effects, and low-latency streaming, making sure sounds are in sync with the action on-screen.

Smart Resource Management. Managing assets like models, textures, shaders, and audio files is hassle-free with Play Engine’s built-in resource manager. It handles reference counting, lazy loading, and dynamic memory allocation to keep things running smoothly and reduce load times.

Intuitive Debugging and UI with ImGui. Debugging and tool development are made easy with the built-in ImGui system. Whether you need real-time debugging panels, in-game controls, or custom interfaces, this system provides a straightforward way to interact with the engine in a visual and intuitive manner.

Flexible Scene Management. The engine's scene system is designed to make structuring and organizing game worlds easy. It supports hierarchical scene graphs, object instancing, and seamless scene transitions, all while optimizing performance through object culling and event-driven updates

With its modular design, efficient rendering, and robust resource management, Play Engine strikes a balance between power and usability, giving developers the tools they need to bring their ideas to life without unnecessary overhead.

Main Features

Seamless 3D Model Loading with TinyObjLoader

Load 3D objects dynamically and add them to a Resource Manager to then be used easily.

Optimized Texture Loading in GNF Format

Dynamic and fast texture loading system that allows quick access to them from the Resouce Manager so they can be applied to objects.

Real-Time Audio Playback with Audio Out 2 and NGS2

3D audio playback system made with both PS5 libraries, both Audio Out 2 and NGS2, and in both cases functional with a WAV file reading system.

Smart Resource Management

Resource Manager which stores all objects at the beginning of the execution using filesystem so that they can be easily accessed later.

Intuitive Debugging and UI with ImGui

Small imgui editor which allows you to make live modifications and save them in a scene system

Flexible Scene Management

Scene system that allows object data to be saved in a .json and then loaded at the start of the program.
