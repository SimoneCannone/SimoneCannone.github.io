About the Project

Mobile game with mechanics similar to Vampire Survivor made by three programmers and several artists.

And published on itch.io: https://jominago.itch.io/proyect-invasion-mobile-game (only available on Android).

In this project, the programmers served as designers but my main task in programming was the implementation of the UI, the weapon and its improvement system and the entire sound section of the game. Apart from helping my colleagues in everything possible in their tasks and solving global problems.

Main Tasks

In this project, I focused on developing the weapon and its leveling system, which is randomized. The system works by presenting three upgrade cards every X number of enemies defeated. Each card offers a different upgrade option, and duplicates are not allowed—meaning the same upgrade cannot appear twice in the same selection. Additionally, at the end of each level, the player is given the choice of a more significant upgrade, which will not appear again once selected.

Besides this, I was also responsible for implementing all the game's sound effects and music, ensuring a cohesive and immersive audio experience. Additionally, I handled the menu management, making sure the user interface was smooth and intuitive.

Main Features

Mobile Game

Game developed so that it is compatible with most Android devices. Performing code and resource optimizations.

Level Up System

Introduced a dynamic leveling system, similar to *Vampire Survivor*, where players gain experience and unlock powerful upgrades after each level-up, enhancing their gameplay experience.

Unity

Developed using Unity to ensure a highly optimized and scalable game engine environment for seamless gameplay.

Compatible with Android

The game is fully optimized for Android devices, offering smooth performance and responsiveness on mobile platforms.
